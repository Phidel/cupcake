## Features

- `游릭` Syntax Highlighting
- `游릭` Auto Indentation
- `游릭` Auto completions
- `游리` Code Debugging
- `游리` Code Folding
- `游댮` Integrated Terminal

## Usage

### Prerequisites

- python
- pip packages
    > Install with `pip install -r requirements.txt`

### Running

- Run `python .` *(don't miss the dot in the end)*
- Open your cpp script.
- Start editing.
