Getting Started

In this section, we will cover the basics of getting started with Cupcake.

Download

In order to go through this section, you should download cupcake from the downloads or github releases.

Installation

Double click the downloaded installer to start the setup of Cupcake. Go through the setup process and then click the finish button. After installation, you can find the Cupcake folder in your windows start menu. You can now launch Cupcake and start coding!

Building From Source

If you wish to build cupcake on your own, you can follow this section. Make sure you have git/github desktop and python installed.

Make sure you have latest python and git installed.
First let's clone the repository from github:
git clone https://github.com/billyeatcookies/cupcake.git
Then let's navigate to the cupcake folder:
cd cupcake
Now run the following command
python .
And that's it! You can now start coding!
